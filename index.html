Коротко о Preact

  • Производительность: быстрый рендеринг и перфоманс
  • Размер: легкий (около 3,5 КБ)
  • Понятность: разобраться в этой библиотеке можно за несколько часов;
  • Совместимость: Preact стремится к тому, чтобы быть максимально совместимым с React API (preact-compat)

Коротко о WebView / WKWebView

  • Это виджет, предоставляемый операционной системой, который позволяет приложениям отображать веб-страницы в приложении.

Преимущества

  • Экономия
  • Кроссплатформенность
  • Гибкость (работает в браузере)

Недостатки

  • Безопасность
  • Отсутствие нативного UI
  • Отсутствие доступа к нативным API
  • Скорость (скроллинг, анимация)
